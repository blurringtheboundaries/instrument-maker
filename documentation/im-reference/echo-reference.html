im.echo

Information:

Echo unit with feedback. This effect is completely wet\, so it's best used as a parallel (send) effect.

Arguments:

delay time in ms, feedback amount as percentage

Inlets:

signal input, delay time, feedback (0-1), clear

Outlets:

processed signal
