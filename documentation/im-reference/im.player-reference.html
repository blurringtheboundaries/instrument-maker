Instrument Maker Reference

im.player

Description:

Retrigger a sound file using an "on" signal (ignores "off"). Save the file you want to trigger in a folder. Add the folder to your pd search path from the preferences menu.

Args:

file name, number of voices (mono by default \, max 4),

Inlets:

on trigger (e.g. im.sensor or im.keypress), off trigger

Outlets:

pulsewave signal
