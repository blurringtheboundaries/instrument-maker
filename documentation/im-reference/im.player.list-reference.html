im.player.list

Information:

Retrigger a sound file using an "on" signal (ignores "off"). Save the list of files you want to trigger as a text file, and use this as the argument.

Arguments:

txt file for playlist, number of voices (mono by default),

Inlets:

audio from the file triggered

Outlets:

sound triggered
