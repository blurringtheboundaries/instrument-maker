Instrument Maker Reference

im.sinewave

Description:

Generate a sine wave (often thought of as a 'pure' tone- like an ocarina or a smooth rumbling sensation at lower frequencies).

Expects audio in the left hand side. Outputs audio.
Args:

note (optional), octave (optional),

Inlets:

pitch from im.scale (range of 0 - 127)

Outlets:

sine wave signal
