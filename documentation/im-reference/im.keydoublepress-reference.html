Instrument Maker Reference

im.keydoublepress

Information:

Double-press a key to trigger an event. Watch out - it doesn't always work to hold more than one key down together. If you find that certain notes hang try pressing the relevant key again.

Arguments:

letter of key to monitor, time to register a second press (ms)

Inlets:

none

Outlets:

off/on signal (0/1)
