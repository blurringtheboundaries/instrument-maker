im.envelope

Description:

This is meant to be an AHR (attack-hold-release) envelope generator. Currently it generates the attack on receiving a positive signal\, and release on zero.

Args:

attack time in ms, release time in ms

Inlets:

gate signal

Outlets:

envelope signal
