Instrument Maker Reference

im.oneshot

Description:

Record like im.looper but play as a oneshot sample. Sounds are saved with the name you give them. Make sure you copy them before you record over them though!

Expects audio in the left hand side. Outputs audio.
Args:

name of audio file

Inlets:

audio input, record control, play control

Outlets:

sample playback
