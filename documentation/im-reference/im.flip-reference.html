Instrument Maker Reference

im.flip

Description:

Assuming a range of 0-1, im.flip inverts its input. Useful for inverting a fader or setting a button trigger when released instead of pressed.

Expects control in the left hand side. Outputs control.
Args:

none

Inlets:

value to flip

Outlets:

flipped value
