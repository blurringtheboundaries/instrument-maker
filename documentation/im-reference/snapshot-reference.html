Instrument Maker Reference

im.snapshot

Description:

Convert a signal from an im object into a message to interface with regular Pd objects. Use this with Pd's number box to observe signals within the patch.

Expects messages in the left hand side. Outputs messages.
Args:

change (optional),

Inlets:

signal

Outlets:

message
