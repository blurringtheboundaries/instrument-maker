Instrument Maker Reference

im.fizzy

Description:

An 80s style synthesiser. One of a set of complex tone generators included in the Instrument Maker library. Fizzy generates a note when it receives a change in signal at its leftmost inlet.

Args:

none

Inlets:

pitch from a scale input, velocity (0-1)

Outlets:

sounds
