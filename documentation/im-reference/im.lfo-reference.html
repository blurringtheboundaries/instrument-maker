Instrument Maker Reference

im.lfo

Description:

Generate a low frequency oscillator to modulate other signals. The LFO is usually not connected directly to the speaker. It should instead be used to change volume\, filter\, pitch\, etc.

Args:

speed (slow/medium/fast), wave (sawtooth/square/sine)

Inlets:

rate (Hz),intensity, reset phase

Outlets:

lfo output (0-1 by default)
