Instrument Maker Reference

im.lessthan

Description:

Send a signal if the input is less than the argument or a signal received in the right-hand inlet.

Expects control in the left hand side. Outputs control.
Args:

number/signal to match (optional)

Inlets:

signal to test, signal to match (if argument is not used)

Outlets:

off/on signal (0/1)
