Instrument Maker Reference

im.greaterthan

Description:

Send a signal if the input is greater than the argument or a signal received in the right-hand inlet.

Args:

number/signal to match (optional)

Inlets:

signal to test, signal to match (if argument is not used)

Outlets:

off/on (0/1)
