Instrument Maker Reference

im.midivelocity

Description:

Break out of the instrument maker objects into MIDI. Experimental/work in progress! Trigger fixed notes with a sensor/switch on/off. Note on velocity is set to a fixed value.

Args:

velocity (optional)

Inlets:

velocity (on/off) input

Outlets:

midi velocity
