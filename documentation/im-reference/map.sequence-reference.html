Instrument Maker Reference

im.sequence

Description:

Play a sequence of notes as specified in the arguemnts. Use [im.step] to convert sensor values into a clock for im.sequence. Try an LFO object to sequence automatically.

Expects control in the left hand side. Outputs control.
Args:

1-8) numbers to output for each corresponding step

Inlets:

audio signal (0-1), signal to record, trigger to record

Outlets:

sequence as audio control signal
