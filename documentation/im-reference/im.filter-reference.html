Instrument Maker Reference

im.filter

Information:

A resonant lowpass filter -- this means that only low frequencies are allowed through. Resonance control boosts the point at which the frequencies are cut off.

Arguments:

cutoff (in Hz), resonance (0-1, or %)

Inlets:

signal to filter, cutoff (0-1), resonance (0-1)

Outlets:

filtered signal
