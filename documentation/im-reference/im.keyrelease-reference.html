Instrument Maker Reference

im.keyrelease

Information:

Release a key to trigger an event. Watch out - it doesn't always work to hold more than one key down together. If you find that certain notes "hang", try pressing the relevant key again.

Arguments:

letter of the key to monitor

Outlets:

off/on signal (0/1)
