Instrument Maker Reference

im.keypress

Description:

Hold keys down to play certain sounds. Watch out - it doesn't always work to hold more than one key down together. If certain notes hang try pressing the relevant key again.

Outputs control.
Args:

letter of the key to monitor

Inlets:

none

Outlets:

off/on signal (0/1)
