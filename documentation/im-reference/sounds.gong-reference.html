Instrument Maker Reference

im.tunedperc

Information:

Play some small synthesised gongs as a standalone synthesiser. The velocity input takes a range of 0-1 or 0-127 (this is detected automatically). If no velocity is received then it defaults to maximum.

Arguments:

none

Inlets:

pitches from im.scale, velocity (0-1 \, or 0-127)

Outlets:

synthesised sound
