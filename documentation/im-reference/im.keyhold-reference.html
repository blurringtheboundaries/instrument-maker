Instrument Maker Reference

im.keyhold

Description:

Hold a key to trigger an event. Watch out - it doesn't always work to hold more than one key down together. If you find that certain notes hang try pressing the relevant key again.

Args:

letter of the key to monitor, hold time threshold in milliseconds

Inlets:

none

Outlets:

off/on signal (0/1)
