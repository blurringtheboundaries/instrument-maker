Instrument Maker Reference

im.midinote

Information:

Break out of the instrument maker objects into MIDI. Velocity is set to a static value (127 by default).

Arguments:

velocity (optional)

Inlets:

pitch input, velocity (on/off) input

Outlets:

midi pitch value, midi velocity
