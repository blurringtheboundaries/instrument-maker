im.electrode

Description:

This was originally a temporary object created for workshop use. The electrodes on the Bare Conductive Touch Board are available if the board is programmed using the midi_interface_generic sketch in their built-in examples.

Args:

electrode number (count from 1)

Outlets:

control signal (0-1)
